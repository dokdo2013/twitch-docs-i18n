# 채팅창 임베드하기

## 사용법

아래 HTML 템플릿을 이용해 트위치 채팅창을 웹사이트에 임베드할 수 있습니다. 채팅창을 설정하기 위한 파라미터는 아래에서 설명합니다.

```html
<iframe src="https://www.twitch.tv/embed/<channel>/chat?parent=<parent>"
        height="<height>"
        width="<width>">
</iframe>
```

## 파라미터

| 이름 | 필수여부 | 설명 |
| --- | --- | --- |
| `src` | 예 | 채팅창 임베드 URL. 이 파라미터는 채팅창을 보여줄 채널과 임베드가 될 도메인을 정의합니다.<br /><br />`channel`: 채널 이름.<br />`parent`: 트위치를 임베드할 도메인. URL에 여러 개의 parent 파라미터를 정의할 수 있습니다. 사이트가 호스팅되거나 임베드되는 각 도메인에 대해 parent 파라미터가 필요합니다. |
| `height` | 아니오 | 채팅창의 높이. 픽셀 단위나 퍼센트 단위로 정의할 수 있습니다. (예: `500` 또는 `50%`) |
| `width` | 아니오 | 채팅창의 너비. 픽셀 단위나 퍼센트 단위로 정의할 수 있습니다. (예: `500` 또는 `50%`) |
| `sandbox` | 아니오 | 모든 트위치 기능이 예상대로 작동하도록 하려면 다음 키워드를 포함해야 합니다: `allow-storage-access-by-user-activation allow-scripts allow-same-origin allow-popups allow-popups-to-escape-sandbox allow-modals` |

## 예제

다음 예제는 TwitchDev 채널의 채팅창을 이 사이트에 임베드하는 방법을 보여줍니다.

소스 코드

```html
<iframe id="twitch-chat-embed"
        src="https://www.twitch.tv/embed/twitchdev/chat?parent=twitch-docs-i18n.vercel.app"
        height="500"
        width="350">
</iframe>
```
