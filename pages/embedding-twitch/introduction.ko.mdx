# 트위치를 웹사이트에 임베드하기

## 소개

다음 세가지 방법을 이용해서 트위치 기능을 웹사이트에 임베드할 수 있습니다:

- [전체 임베드하기](/ko/embedding-twitch/everything) 는 라이브 방송이나 영상을 재생할 수 있는 비디오 플레이어와 트위치 채팅을 하나의 iframe으로 임베드하는 방법을 설명합니다. 여기에는 팔로우, 구독, 그리고 로그인 기능이 포함됩니다.
- [채팅창 임베드하기](/ko/embedding-twitch/chat) 는 트위치 채팅창을 임베드하는 방법을 설명합니다.
- [비디오 & 클립 임베드하기](/ko/embedding-twitch/video-and-clips) 는 비디오 플레이어와 클립을 임베드하는 방법을 설명합니다.

## 임베드를 위한 요구사항

트위치를 임베드하기 위해서는 반드시 아래의 요구사항과 트위치 개발자 서비스 약관을 준수해야 합니다. 트위치는 요구사항을 준수하지 않는 도메인에 대해 임베드 사용을 제한할 수 있습니다. 트위치는 언제든지 어떤 이유로든 임베드 사용을 취소할 권리를 보유합니다.

1.1 트위치 임베드를 사용하는 도메인은 SSL 인증서를 사용해야 합니다.

1.2 트위치는 웹 상에서 트위치 제품을 사용하는 유저를 인증하기 위해 `parent`라는 특별한 파라미터를 요구합니다. 이 파라미터는 임베드 방식에 따라, iframe `src` 속성의 쿼리 파라미터로 제공하거나, 트위치 임베드를 생성하는 자바스크립트 객체에 속성(property)으로 제공해야 합니다. 자세한 내용은 위의 임베드 API 문서와 요구사항에 대한 초기 공지를 참고하세요. 이 파라미터를 지정하지 않은 도메인은 재생 오류 메시지를 표시하고, 트위치는 해당 콘텐츠 시청을 원하는 유저가 트위치로 이동하도록 유도합니다.

1.3 임베드는 반드시 인증된 트위치 플레이어만을 사용해야 합니다. 또한 임베드는 도메인 내에 표시되는 페이지 요소 등으로 가려지지 않아야 합니다. 채팅 메시지 전송과 같은 특정 기능은, iframe이 가려지거나 보이지 않는 경우 비활성화될 수 있습니다.

1.4 임베드는 속성 테이블에 표시된 최소 높이와 너비 요구사항을 반드시 준수해야 합니다.
