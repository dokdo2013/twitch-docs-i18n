# 개요

다국어를 지원하는 트위치 API 문서 프로젝트에 오신 걸 환영합니다! 이 커뮤니티 기반 프로젝트는 트위치 API를 사용하는 전세계의 개발자들이 더 쉽게 문서에 접근할 수 있도록 번역본을 제공하는데 초점을 두고 있습니다. Next.js로 구축된 모던 정적 웹사이트 생성기 Nextra를 이용해, 보다 유저 친화적이고 쉽게 유지보수 가능한 문서를 유지합니다.

## 특징

- 트위치 API 문서의 커뮤니티 기반 번역
- 쉬운 페이지간 이동과 검색 경험 제공
- Next.js 기반 Nextra로 제작되어 개선된 웹사이트 경험 제공
- 오픈소스 프로젝트로 전 세계 개발자들의 기여

## 시작하기

본 프로젝트를 시작하기 위해선, 간단하게 본 레포지토리를 복제하여 README 문서에 기재된 방법을 따라가면 됩니다. 문서를 번역하거나 번역된 문장을 개선하는 방식으로 프로젝트에 기여할 수 있습니다.

## 기여하기

우리는 개발자들이 가진 모든 배경지식과 기술을 이용해 본 프로젝트에 기여하는 것을 장려합니다. 여러 언어에 능통하거나, 아니면 막 시작하는 단계이더라도, 여러분의 기여는 트위치 API에 대한 전세계적 접근성을 향상시키는데 도움을 줄 것입니다.
