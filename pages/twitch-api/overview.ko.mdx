# 트위치 API

트위치 API는 트위치와 연동하기 위한 도구와 데이터를 제공합니다. 데이터 모델과 시스템은 쉽고 일관되며 신뢰할 수 있는 방식으로 관련 데이터를 제공하도록 설계되었습니다. 연동에 사용할 수 있는 엔드포인트의 전체 목록은 [트위치 API 레퍼런스](/ko/twitch-api/reference)를 참조하세요.

트위치 API는 인증을 위해 OAuth 2.0을 사용합니다. API가 지원하는 여러 종류의 액세스 토큰에 대해 더 자세히 알고 싶다면, [인증](/ko/authentication/overview) 메뉴를 살펴보세요. 확장 프로그램과 관련된 엔드포인트를 이용할 계획이 있다면, JSON 웹 토큰(JWT)을 얻는 방법도 알아야 합니다. (자세한 내용은 [JSON 웹 토큰](/ko/extensions/required-technical-background#json-web-tokens-jwt)과 [확장 프로그램 암호 관리](/ko/extensions/building#managing-extension-secrets)를 참조하세요.)

다음 기능에 대해서도 알아두는 것이 좋습니다.

| 기능 | 설명 |
| - | - |
| [EventSub](/ko/eventsub/overview) |	트위치 API는 특정 리소스의 상태를 주기적으로 조회하기 위한 API를 제공합니다. 이 API는 리소스의 상태를 기록하기 위한 용도로 사용할 수 있지만, 리소스 변경에 대한 구독을 더 추천합니다. EventSub 구독에 대한 자세한 내용은 [EventSub](/ko/eventsub/overview)을 참조하세요. |
| [명령 줄 인터페이스](/ko/twitch-cli/overview) |	트위치는 트위치 리소스를 관리하기 위한 명령 줄 인터페이스를 제공합니다. 이를 사용하여 API 엔드포인트를 호출하고, OAuth 액세스 토큰을 얻으며, EventSub 이벤트를 테스트할 수도 있습니다. |
