# 레퍼런스

## 송출 서버 조회

송출 서버 조회는 Twitch로 라이브 비디오를 송출하기 위한 엔드포인트 목록을 반환합니다.

**URL**

`GET https://ingest.twitch.tv/ingests`

**인증**

없음

**응답 값**

| 매개변수 | 타입 | 설명 |
| --- | --- | --- |
| `ingests` | array | 송출 서버 객체의 배열입니다. |
| `_id` | integer | 송출 서버의 순번입니다. |
| `availability` | float | 내부적으로 사용되는 예약 값입니다. |
| `default` | boolean | 내부적으로 사용되는 예약 값입니다. |
| `name` | string | 송출 서버의 이름입니다. |
| `url_template` | string | 송출 서버의 RTMP URL 템플릿입니다. |
| `priority` | integer | 내부적으로 사용되는 예약 값입니다. |

**요청 예시**

```bash
curl -X GET 'https://ingest.twitch.tv/ingests'
```
